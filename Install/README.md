# Missing Image Tool - Installation Files

This folder contains the installation files for the B2C Commerce Missing Image Analyzer.

## Installation Instructions

### For macOS Users
**File:** `Missing Image Tool-1.0.0.dmg`

1. Double-click the `.dmg` file
2. Drag "Missing Image Tool" to your Applications folder
3. Launch from Applications or Launchpad

**Note:** On first launch, you may need to right-click the app and select "Open" to bypass macOS security warnings.

### For Windows Users
**File:** `Missing Image Tool Setup 1.0.0.exe`

1. Double-click the `.exe` file
2. Follow the installation wizard
3. Launch from Start Menu or Desktop shortcut

**Note:** Windows Defender may show a warning since the app is not signed. Click "More info" then "Run anyway" to proceed.

## What's New in Version 1.0.0

- ✅ **Online Products Only** - Automatically filters to only analyze online products (online-flag="true")
- ✅ **Optional View Type** - Leave view type empty to check ALL image view types, or specify one to check only that type
- ✅ **Category Filtering** - Only analyzes products assigned to at least one category
- ✅ **WebDAV Verification** - Checks if image files actually exist on your server
- ✅ **No File Size Limits** - Handle catalogs of any size with streaming parsers
